# Figure 1 — 2025 Northwestern Pacific Summer Marine Heatwave Overview

This folder contains the code, derived data, and final figure for **Figure 1** of the paper:

> **Role of Western Tibetan Plateau Thermal Forcing in the 2025 Summer Extreme Marine Heatwave over the Northwestern Pacific**
>
> Jianning Tao¹, Wei Hua¹\*, Lihua Zhu¹, Xiaofei Wu¹, Shangru Li¹
>
> ¹ School of Atmospheric Sciences / Climate Change and Resource Utilization in Complex Terrain Regions Key Laboratory of Sichuan Province / Sichuan Provincial Engineering Research Center for Meteorological Disaster Prediction and Early Warning, Chengdu University of Information Technology, Chengdu, China
>
> \* Corresponding author: Wei Hua

Figure 1 is a six-panel (3×2) overview of the 2025 summer (JJA) marine heatwave (MHW) over the
Northwestern Pacific and a first-order diagnosis of the surface heat budget driving it.

![Figure 1](figures/sst_6panel_combined_new.png)

---

## Panels

| Panel | Content | Source |
|-------|---------|--------|
| **(a)** | 2025 JJA sea-surface-temperature anomaly (SSTA) map; green box marks the analysis region | gridded detrended SST |
| **(b)** | 2025 JJA SST time series for the box region, with MHW climatology / 90th-percentile threshold and event shading | `marineHeatWaves` detector |
| **(c)** | Standard deviation of JJA SSTA, 1982–2025 (interannual variability) | gridded detrended SST |
| **(d)** | Box-mean JJA SSTA, 1982–2025: raw vs. de-trended | `data/sst_panel_d_raw_vs_detrended_1982_2025.csv` |
| **(e)** | Mixed-layer heat budget for 2025 JJA: ΔSSTₐ, Qnet term, advection term, residual | `data/tp_heat_budget_2025_summer_terms.csv` |
| **(f)** | Decomposition of the surface net heat flux (Qnet) into shortwave, longwave, sensible, latent contributions | `data/tp_qnet_component_contribution_2025.csv` |

## How Figure 1 is built

`plot_sst_6panel_combined.py` keeps panels **(a)–(d)** identical to the 4-panel overview and adds
panels **(e)** and **(f)** below them:

- Panels (a), (c), (d), (e) reuse routines from the bundled base module `plot_sst_2x2_combined.py`
  (`compute_map_panel_data`, `compute_panel_c_data`, `compute_panel_d_budget`), which the main
  script imports at runtime.
- Panel (b) is computed in the main script via the `marineHeatWaves` event detector.
- Panel (f) is computed in the main script (`compute_panel_f_qnet_components`) from ERA5 net-flux
  components and a GODAS-derived mixed-layer depth.

## Requirements

- Python ≥ 3.10
- `numpy`, `pandas`, `scipy`, `xarray`, `matplotlib`, `cartopy`, `cmaps`
- [`marineHeatWaves`](https://github.com/ecjoliver/marineHeatWaves) — Hobday et al. (2016) MHW detector

> **Note:** this study ran against a copy of `marineHeatWaves.py` hand-patched for NumPy 2.x
> compatibility. A stock install on NumPy 2.x may need the same patch (e.g. replacing removed
> `np.float`/`np.int` aliases) for `mhw.detect()` to run.

## Reproducing the figure

The two scripts use **absolute paths** pointing at the original compute environment (both for the
input datasets and for the `importlib` load of the base module). To regenerate Figure 1 elsewhere:

1. Place `plot_sst_6panel_combined.py` and `plot_sst_2x2_combined.py` in the same directory.
2. In `plot_sst_6panel_combined.py`, set `BASE_SCRIPT_PATH` to the local path of
   `plot_sst_2x2_combined.py`, and update `OUT_DIR` to your output directory.
3. Update the input-data paths (below) in both scripts to your local copies.
4. Run:

   ```bash
   python plot_sst_6panel_combined.py
   ```

   This writes `sst_6panel_combined_new.png` (600 dpi) plus the three CSVs in `data/`.

## Key region

The analysis box (green box in panel a) covers the Northwestern Pacific MHW region:
~27–48°N, 127–164°E.
